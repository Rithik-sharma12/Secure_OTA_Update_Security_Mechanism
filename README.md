# Secure_OTA_Update_Security_Mechanism
OTA_IOT is a comprehensive research and development project focused on building secure Over-The-Air (OTA) firmware update systems for heterogeneous IoT devices. The project is titled "SentinelOTA: Building Resilient IoT Defense".

---

## 🔄 Syncing with the Repository

### Clone the repo (first time only)
```bash
git clone https://github.com/Rithik-sharma12/Secure_OTA_Update_Security_Mechanism.git
cd Secure_OTA_Update_Security_Mechanism
```

### Pull latest changes from remote to local
```bash
git pull origin main
```

### Push your local changes to remote
```bash
git add -A
git commit -m "Your commit message here"
git push origin main
```

### If you get a conflict when pushing (someone else pushed before you)
```bash
git pull --rebase origin main
git push origin main
```

### Quick Reference

| Task | Command |
|---|---|
| Get latest changes | `git pull origin main` |
| Stage all changes | `git add -A` |
| Commit changes | `git commit -m "message"` |
| Push to remote | `git push origin main` |
| Check status | `git status` |
| View commit history | `git log --oneline` |
